<h1>Chat Interativo</h1>

<h2>Estrutura do Projeto</h2>

+ <b>HTML:</b> O arquivo index.html contém a estrutura da página, incluindo um campo de entrada de texto para respostas do usuário, um botão de envio e uma área para exibir as conversas.
+ <b>CSS:</b> O arquivo estilo.css contém os estilos para a página, incluindo a largura da janela do chat, o estilo do botão e o formato das mensagens exibidas.
+ <b>JavaScript:</b> O arquivo scripts.js contém o código JavaScript responsável pela lógica do chat, como exibir mensagens de boas-vindas, opções de escolha e respostas do usuário.

<h2>HTML (index.html)</h2>
O arquivo HTML é simples e contém os elementos necessários para o funcionamento básico do chat, como a entrada de texto, botão de envio e área de exibição das conversas.

<h2>CSS (estilo.css)</h2>
Os estilos CSS são aplicados para melhorar a aparência e a usabilidade do chat. Ele define a largura da janela do chat, o estilo do botão de envio e o formato das mensagens exibidas.

<h2>JavaScript (scripts.js)</h2>
O JavaScript é responsável por implementar a lógica do chat, como exibir mensagens de boas-vindas, apresentar opções de escolha e responder às entradas do usuário.

<h2>Funcionamento</h2>

+ Ao abrir a página, uma mensagem de boas-vindas é exibida automaticamente após 2 segundos.
+ O usuário pode inserir sua resposta no campo de entrada de texto e clicar no botão "Enviar".
+ Dependendo da resposta do usuário, o chat exibe uma mensagem correspondente com base nas opções disponíveis.
  
<h2>Personalização</h2>
Você pode personalizar o chat alterando os estilos no arquivo estilo.css e a lógica de resposta no arquivo scripts.js para atender às suas necessidades específicas.
